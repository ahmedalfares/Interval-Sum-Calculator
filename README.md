# Interval-Sum-Calculator

A simple C-based utility that calculates the summation of numbers within specified intervals provided by the user.

## Features
The program takes three inputs (X, Y, Z) in ascending order and computes:
* **Even Sum:** The total of all even numbers between **X** and **Y**.
* **Odd Sum:** The total of all odd numbers between **Y** and **Z**.

## How to Run
1. **Compile the source code:**
   ```bash
   gcc main.c -o IntervalSum
2. Excute program

Example output:
ENTER VALUES FOR X Y Z (ASCENDING):
2 10 15
